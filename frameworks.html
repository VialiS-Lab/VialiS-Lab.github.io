🧩 Frameworks

Human-AI Engineering 시리즈가 정의하는 6개의 독자적 프레임워크입니다. 각 프레임워크는 해당 시리즈에서 실습을 통해 적용합니다.

Phase 1

AI OS · EBRE

→ Working with AI 시리즈에서 배우기

파일 기반 AI Workspace로 개인 업무 운영체제를 구축하고, 완성된 시스템을 증거 기반으로 역공학하여 패턴과 설계 원칙을 추출하는 방법론입니다.

  • AI OS — AI_Work/ 폴더, AI_RULES.md, 3단계 QC(기계·논리·인간), 워크플로우(재료→가공→결과물), GLOBAL_RULES.md, 자가 치유(재시도·에스컬레이션)
  • EBRE(Evidence-Based Reverse Engineering) — 관찰 → 코딩 → 패턴 → 아키텍처 4단계로 자신의 시스템을 분석

Phase 2

QHER Engine

→ Thinking with AI 시리즈에서 배우기

Question → Hypothesis → Evidence → Reasoning의 4단계 사고 파이프라인으로, Decision과 Synthesis는 Reasoning 내부 프로세스로 통합됩니다.

  • EQI(Evidence Quality Index) — 출처·직접성·표본·재현성 가중합으로 증거 품질을 0~5점으로 환산
  • Reasoning Audit — 주장·가정·증거매핑·누락증거·반박·신뢰도 6단계 감사 체크리스트
  • Auto-Rules(AR-1~4) — 각 단계의 출력을 다음 단계 입력으로 자동 변환하는 파이프라인 규칙

Phase 3

K-OS Engine

→ Knowledge with AI 시리즈에서 배우기

Capture→Validate→Structure→Connect→Reuse→Evolve→Archive→Retire의 8단계 Lifecycle로 정보를 지식으로 전환하고 관리합니다.

  • EIQ — 출처·신선도·깊이·관련성·재사용성·참신성 6대 기준으로 정보 품질 평가
  • Knowledge Graph 심화 — Hub·Bridge·Orphan·Cluster·Mental Model 발견
  • Knowledge Evolution — Merge·Split·Replace·Fork·Refactor 버전 관리
  • Health Check — Knowledge Density·Reuse Rate·Growth Rate·Health Score

Phase 4

L.E.A.R.N. Engine

→ Learning with AI 시리즈에서 배우기

Locate→Acquire→Retrieve→Next(Apply/Transfer)→Reflect의 인지과학 기반 6단계 학습 파이프라인으로, VARK를 대체합니다.

  • A-P-A-R 모델 — Acquire·Practice·Apply·Retrieve
  • Forgetting Curve + Spaced Repetition — 1·3·7·14·30일 간격 복습
  • Mastery Score / Learning Health — 학습 성과와 시스템 건강도 정량화
  • AI 5대 역할 — Tutor·Coach·Examiner·Mentor·Simulator

Phase 5

C.E.N.G.I.N.E. + MIDAR

→ Cognitive Engineering with AI 시리즈에서 배우기

6개 설계 차원(C.E.N.G.I.N.E.)과, 이를 관장하는 메타 인지 제어 루프(MIDAR)로 인간-AI 통합 시스템을 설계합니다.

  • C.E.N.G.I.N.E. — Context·Environment·Neuro-Adaptive·Governance·Interaction·Next-Gen
  • MIDAR — Monitor·Interpret·Decide·Act·Review 순환 루프
  • CSH / HAIS — Cognitive System Health, Human-AI Integration Score 최상위 KPI
  • Adaptive Cognitive States — Creative·Analytical·Decision·Review 모드 자동 전환
  • Policy Framework — Decision·Verification·Escalation·Privacy·Audit 5대 정책
  • Closed Human-AI Loop / Self-Optimization — 순환형 상호작용 + KPI 기반 자동 개선
