L.E.A.R.N. Engine — 학습 시스템

📐 L.E.A.R.N. 프레임워크 (최종 정의)

L.E.A.R.N.은 Learning Engine for Adaptive & Reflective Network의 약자로, 인지과학 기반 6단계 학습 파이프라인입니다.

```

│                         L.E.A.R.N. FRAMEWORK v2.0                              │
```

🛠️ 도구: A-P-A-R 학습 모델 (VARK 대체)

VARK(Visual·Auditory·Reading·Kinesthetic)는 근거가 부족한 모델입니다. 대신 인지과학 기반 A-P-A-R 모델을 사용합니다.

단계설명AI 활용 전략
Acquire (획득)새로운 정보를 입력받는 단계 (읽기·듣기·관찰)AI에게 요약, 구조화, 다이어그램 생성 요청
Practice (연습)정보를 처리하고 이해하는 단계 (설명·재구성·질문 생성)AI에게 "내가 이해한 것을 설명할게, 확인해줘" 요청
Apply (적용)배운 것을 실제 상황에 사용하는 단계AI와 함께 시뮬레이션, 역할극, 케이스 스터디 실행
Retrieve (인출)기억을 꺼내는 단계 (퀴즈·시험·설명·가르치기)AI가 질문 생성, 내가 답변, AI 채점 및 오답 설명

🛠️ 도구: Forgetting Curve & Spaced Repetition

Learning의 핵심은 망각과의 싸움입니다. Ebbinghaus의 Forgetting Curve를 기반으로 한 복습 시스템입니다.

Forgetting Curve 기본 원리

```
기억률
100% ──────────────────────────────────────────────────
  │  ● (즉시: 100%)
  │    ╲
  │      ╲  ● (1시간: 60%)
  │        ╲
  │          ╲  ● (1일: 40%)
  │            ╲
  │              ╲  ● (3일: 30%)
  │                ╲
  │                  ╲  ● (7일: 25%)
  │                    ╲
  │                      ╲  ● (30일: 20%)
  └─────────────────────────────────────────────────────▶ 시간
```

Spaced Repetition 복습 간격

복습 회차최적 간격활동
1즉시 (학습 직후)Retrieve Practice (AI 질문 → 답변)
21일 후Retrieve + 오답 재학습
33일 후Retrieve + K-OS 연결 확인
47일 후Apply (실제 업무 적용 시도)
514일 후Retrieve + Transfer (다른 분야 적용)
630일 후Mastery Score 재측정

🛠️ 도구: Mastery Score & Learning Health

Learning의 성과를 정량적으로 측정하는 지표입니다.

Mastery Score (숙련도 점수)

```
Mastery Score = (Recall × 0.40) + (Application × 0.30) + (Transfer × 0.20) + (Speed × 0.10)
```
항목측정 방법
Recall (기억)AI가 생성한 5개 질문에 정답률
Application (적용)실제 업무 적용 성공률 (자기 평가)
Transfer (전이)다른 상황/분야에 적용한 횟수 및 성공률
Speed (속도)과제 수행 시간 (학습 전 vs 후)

Mastery Score 해석

점수등급의미
90~100⭐⭐⭐⭐⭐ Expert가르칠 수 있는 수준
75~89⭐⭐⭐⭐ Advanced타인에게 설명 가능
50~74⭐⭐⭐ Intermediate혼자서 적용 가능
25~49⭐⭐ Beginner기본 개념 이해
0~24⭐ Novice학습 필요

Learning Health Score (학습 건강 지수)

```
Learning Health = (Mastery × 0.30) + (Retrieval × 0.25) + (Transfer × 0.20) + (Reflection × 0.15) + (Consistency × 0.10)
```
항목의미
Mastery현재 Mastery Score 평균
Retrieval최근 5회 Retrieve 정답률
Transfer학습 내용을 다른 분야에 적용한 횟수
Reflection성찰 노트 작성 빈도 및 깊이
Consistency학습 일관성 (주간 학습 시간)

🛠️ 도구: AI 역할 체계 (Tutor·Coach·Examiner·Mentor·Simulator)

Learning with AI에서 AI는 단순 도구가 아니라 5가지 역할을 수행합니다.

역할기능사용 시점프롬프트 예시
Tutor개념 설명, 예시 제공, 질문 답변Acquire 단계"이 개념을 초보자에게 설명하듯 알려줘"
Coach학습 계획 수립, 동기 부여, 피드백Locate + Assess"내 목표에 맞는 2주 학습 계획을 만들어줘"
Examiner퀴즈 생성, 채점, 오답 설명Retrieve 단계"이 주제로 5개 질문을 만들어줘. 내 답변을 채점해줘"
Mentor경험 공유, 조언, 인사이트 제공Reflect 단계"이 주제에서 가장 중요한 인사이트는 무엇일까?"
Simulator가상 상황, 역할극, 케이스 스터디Next/Apply 단계"이 상황에서 어떻게 적용할지 시뮬레이션해줘"

📌 Module 1: 배울 주제를 선정하자 (Locate)

L.E.A.R.N.의 첫 번째 단계. 무엇을, 왜 배울 것인지를 정의하고, 학습 목표를 명확히 설정합니다.

① Experiment — 목표 없는 학습 vs 목표 있는 학습

당신이 해야 할 일: 두 가지 방식으로 AI에게 학습을 요청해보세요.

방식 A (목표 없음):

"AI에 대해 알려줘."

방식 B (목표 있음 - Learning Goal Matrix 적용):

"나는 AI를 업무에 활용하려는 마케터입니다. 이번 학습을 통해 ① AI 프롬프트의 기본 구조를 이해하고(지식), ② 실제 마케팅 카피를 작성할 수 있으며(기술), ③ AI에 대한 두려움을 극복하고(태도), ④ 주간 보고서 작성 시간을 30% 단축하는 것(성과)이 목표입니다. 이 목표에 맞는 학습 로드맵을 제안해주세요."

→ 어떤 요청이 더 구체적이고 실행 가능한 학습 계획을 주었나요?

② Reflection — 생각해보기

  • 당신은 학습을 시작할 때 구체적인 목표를 설정하나요?
  • 목표가 없을 때와 있을 때, 학습의 집중도와 결과는 어떻게 달라지나요?

③ Discovery — 방금 깨달은 것

"아, '무엇을 배울까?'보다 '왜 배우고, 배운 후에 무엇이 달라질까?'가 훨씬 중요하구나."

④ Rule — L.E.A.R.N. 법칙

법칙내용
Rule L1학습은 목표에서 시작한다. 목표가 없는 학습은 방향을 잃은 탐험과 같습니다.
Rule L2좋은 학습 목표는 4가지 차원을 포함한다. ① 지식(무엇을 알게 될까), ② 기술(무엇을 할 수 있을까), ③ 태도(무엇이 달라질까), ④ 성과(어떤 결과가 나올까).
Rule L3Mastery Score 목표를 함께 설정하라. "이 주제를 80점 이상으로 마스터하겠다"는 구체적인 기준이 학습을 강화합니다.

📌 Module 2: 지식을 획득하자 (Acquire)

L.E.A.R.N.의 두 번째 단계. A-P-A-R 모델의 Acquire(획득) 단계로, 새로운 정보를 입력받고 이해합니다.

① Experiment — Acquire 방식 비교

당신이 해야 할 일: 같은 주제를 세 가지 방식으로 획득(Acquire)해보세요.

방식 1 - 읽기: 관련 아티클이나 문서를 읽어보세요.
방식 2 - 듣기: 관련 강의나 팟캐스트를 들어보세요.
방식 3 - 관찰: 실제 사례나 데모를 관찰해보세요.

→ 어떤 방식이 가장 빠르게 이해되었나요?

② Reflection — 생각해보기

  • Acquire 단계에서 가장 중요한 것은 무엇인가요? (속도? 깊이? 정확성?)
  • AI(Tutor)에게 어떤 도움을 받을 수 있을까요?

③ Discovery — 방금 깨달은 것

"아, 획득(Acquire)은 정보를 받아들이는 단계야. 내가 어떻게 받아들이느냐에 따라 이후 학습이 완전히 달라지네."

④ Rule — L.E.A.R.N. 법칙

법칙내용
Rule A1Acquire는 학습의 재료를 모으는 단계다. 좋은 재료가 좋은 학습을 만듭니다.
Rule A2AI Tutor는 Acquire를 도와주는 최고의 도구다. 요약, 구조화, 시각화, 질문 생성을 요청하세요.
Rule A3Acquire 후에는 반드시 1차 정리를 하라. 배운 내용을 3문장으로 요약하는 습관을 들이세요.

Step 3. AI의 결과물을 바탕으로, 자신만의 3문장 요약을 작성하세요.
